# Two Player Math Game
![Ruby](https://img.shields.io/badge/Ruby-CC342D?style=for-the-badge&logo=ruby&logoColor=white)

Welcome to 'Two Player Math Game'! Two players must take turns to solve quick addition problems. Each turn, a number between 1 and 20 is given by a player, which prompts a new question. They must answer correctly to stay in the game - a wrong answer costs a precious life. Lose all three and the game is over!
